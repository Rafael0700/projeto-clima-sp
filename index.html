💾 Análise com SQL

Para este projeto, os dados foram tratados como se estivessem armazenados em um banco SQL. Abaixo estão algumas consultas utilizadas:

Consulta para Temperatura Média Mensal


SELECT 
    YEAR(data) AS ano,
    MONTH(data) AS mes,
    AVG(temperatura_media) AS temp_media
FROM clima_sao_paulo
WHERE data BETWEEN '2010-01-01' AND '2020-12-31'
GROUP BY YEAR(data), MONTH(data)
ORDER BY ano, mes;

Consulta para Análise de Correlação


SELECT 
    CORR(temperatura_media, umidade) AS correlacao_temp_umidade,
    CORR(temperatura_media, precipitacao) AS correlacao_temp_chuva
FROM clima_sao_paulo
WHERE data BETWEEN '2010-01-01' AND '2020-12-31';
